# guixu

业务场景的数据关系服务， 通过定义一个模型，管理数据的约束和实例管理。 


## 数据约束

- 字段类型，是否必填，是否允许编辑，是否对用户可见
- 唯一索引
- 实例删除规则
- 是否为数组

## 数据类型

- 字符串
- 长字符串
- 超链接
- 数字
- bool
- enum
- 日期
- 时间
- 外键
- 引用

每一种类型有一个单独校验规则

## 关联关系支持

支持关系：
- 1:1
- 1:N
- N:N

方向：
- 支持单向/双向关系
- parent/next/sub

## 数据类型
- 模型
- 实例
- 关系

### 模型
包含下面数据：
- 模型的字段
- 唯一
- 关系定义
-  分类和字段分组
-  扩展功能： 字段排序，默认展示字段
### 实例
模型定义字段的规则实例化的数据。 

### 关系
根据关系实现层级结构，树状结构及离散的拓扑结构


### 实例
根据模型实例化出来的数据
